Pembahasan HTML

HTML Adalah Hyper Text Markup Language, Bahasa markup dari website.

1. Sejarah HTML

HTML dibuat oleh seorang yg bernama Tim Berners Lee Hypertext Markup Language (HTML) adalah sebuah bahasa markah yang digunakan untuk membuat sebuah halaman web, menampilkan berbagai informasi di dalam sebuah penjelajah web Internet dan pemformatan hiperteks sederhana yang ditulis dalam berkas format ASCII agar dapat menghasilkan tampilan wujud yang terintegrasi. Dengan kata lain, berkas yang dibuat dalam perangkat lunak pengolah kata dan disimpan dalam format ASCII normal sehingga menjadi halaman web dengan perintah-perintah HTML. Bermula dari sebuah bahasa yang sebelumnya banyak digunakan di dunia penerbitan dan percetakan yang disebut dengan SGML (Standard Generalized Markup Language), HTML adalah sebuah standar yang digunakan secara luas untuk menampilkan halaman web. HTML saat ini merupakan standar Internet yang didefinisikan dan dikendalikan penggunaannya oleh World Wide Web Consortium (W3C). HTML dibuat oleh kolaborasi Caillau TIM dengan Berners-lee Robert ketika mereka bekerja di CERN pada tahun 1989 (CERN adalah lembaga penelitian fisika energi tinggi di Jenewa) https://id.wikipedia.org/wiki/HTML


2. Cara Kerja HTML

File HTML diakhiri dngan ekstensi .html File ini bisa dibuka menggunakan web browser apapu. Browser membacafilehtml dan mernder konten di dalam filenya sehingga kode dari dalam file html bisa dibaca jelas d browser oleh user.

Source code ata coding yg dibuat didalam html, diterjemahkan oleh browser melalui prosees interpreted. proses interpreted adalah programmer menulis coding, disimpan ke dalam file html, di file html tersebutlah dibuka ke browser.


3. Syntax HTML

        <!DOCTYPE html>
        <html>
            <head>
                <title>
                </title>
            </head>
            <body>
            </body>
        </html>

Struktur dasar html terdiri dari yg pertama adalah tag <!DOCTYPE html> merupakan tag awal penanda bahwa kita akan menggunakan html.

Selanjutnya tag <html> </html> merupakan tag pembungkus file html, diantara tag pembuka dan penutupnya, disinilah kita dapat menuliskan judul, link penghubung, serta source code dan banyak lagi.

tag <head> </head> merupakan tag yg output nya ditampilkan di judul browser. kita dapat menulis judul website kita di sini. Dalam tag head terdapat tag yg wajib yaitu <title> </title> disinilah judul ditulis, terdapat juga tag yg menyesuaikan ditulis yaitu seperti <link> dalam tag ini digunakan untuk menghubungkan antara html dan css

tag <body> </body> Didalam sinilah kita banyak mengunakan fungsi file html. Dari mulai menulis source code sampai menghubungkan file lainnya

Tag HTML terdapat 2 jenis, Yakni tag html yg mempunyai pasangan dan tag html yg tidak mempunyai pasangan
